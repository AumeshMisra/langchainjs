# Cohere

:::info
This is the documentation for the Cohere provider in the LangChain JS documentation.
:::

## Introduction

Cohere is a company that provides natural language processing (NLP) tools and services, including language models that can be used for a variety of applications such as text generation, classification, and sentiment analysis.

## Installation and Setup

To use Cohere with LangChain JS, you need to install the SDK and set up your API keys.

```bash
npm install langchain-cohere
```

Obtain an API key from Cohere and set it as an environment variable:

```bash
export COHERE_API_KEY='your-api-key'
```

## Quick Copy Examples

Here are some quick examples to get you started with Cohere integrations.

### Text Generation

```javascript
import { TextGeneration } from 'langchain-cohere';

const textGen = new TextGeneration();
const generatedText = textGen.generateText('Once upon a time');
console.log(generatedText);
```

### Classification

```javascript
import { Classification } from 'langchain-cohere';

const classifier = new Classification();
const categories = classifier.classifyText('This is a test sentence.');
console.log(categories);
```

### Sentiment Analysis

```javascript
import { SentimentAnalysis } from 'langchain-cohere';

const sentiment = new SentimentAnalysis();
const sentimentResult = sentiment.analyzeSentiment('I love this product!');
console.log(sentimentResult);
```
